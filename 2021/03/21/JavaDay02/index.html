总结：

  • 整数字面值默认使用4字节的int型来保存

  • 小数字面值默认使用8字节的double型来保存

  • 范围小的值给范围大的变量赋值时，可以直接自动完成

  • 范围大的量值给范围小的变量赋值时，不可以直接完成，必须强制类型转换(强制类型转换有风险！！！)

tip：

  • \1. 兼容性最好的时double，最差的时byte

  • \2. 范围从小到大排序：byte < short < int < long < float < double

  • \3. 任意非long整数作运算，结果总是int型

  • \4. 如果有不同类型的混合运算，结果总是范围大的那个类型。
