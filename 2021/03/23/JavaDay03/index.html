1.变量

总结：

  • 变量按照数据类型来分：
    1.基本数据类型：内存中存储的是数据本身
    2.引用数据类型：内存空间中存储的是别的数据的地址，内存中某个字节的编号

  • 按照变量声明位置来分：
    1.局部变量：声明在方法中的变量，范围小，寿命短
    2.成员变量：声明在类中方法外的变量，范围大，寿命长

2.程序流程控制：

  • 2.1 程序流程控制分为：

    1.顺序结构：程序从上到下逐条执行，中间没有任何判断和跳转
    2.分支结构：有条件，选择性地执行某段代码。 eg：if ···· else 和 swich两种语句
    3.循环结构：根据循环条件，重复执行某段代码。有 while ， do ··· while ， for三种循环语句 ，也有foreach循环，方便遍历集合和数组元素

2.2 总结：

  1. switch中表达式的返回值必须是下述几种之一：byte short char int string 枚举
  2. case子句中的值必须是常量
  3. default子句可任选，没有匹配case，执行default
  4. 没有break语句，会出现穿透现象，就不是严格意义上的分支。
